    使用 Python

    以下是运行 Python 文件中最常用的方式：

    1. 不使用如何命令行选项去运行你提供的文件中的代码，然后返回到命令行。如果你的文件只包含了一个函数定义，除非你有一个语法错误你将不会看到任何输出。

    python3 lab00.py
    2. -i：-i 选项可以在运行你提供的文件中的代码后，进入交互式环境（伴随着一个 >>> 提示符）。你可以进行表达式求值，比如调用你已定义的函数等。可以通过 exit() 来退出。你也可以在 Linux/Mac 上使用键盘快捷键 Ctrl-D 或在 Windows 上使用 Ctrl-z Enter 来进行退出。如果你在运行着交互环境时编写了 Python 文件，你需要退出并重启解释器才能使更改生效。

    以下是我们应该如何在交互环境中运行 lab00.py： 

    python3 -i lab00.py
    3. -m doctest：运行文件中的文档测试，即函数中的文档字符串。每个测试都由 >>> 和一些 Python 代码以及期望的输出组成。

    以下是我们应该如何运行 lab00.py 中的文档测试： 

    python3 -m doctest lab00.py

    当我们的代码通过了所有的测试时，不会有任何输出。否则，会有测试失败的信息显示。

    使用 OK 在 CS 61A 中，我们使用一个称为 Ok 的程序自动测试实验、作业和项目。当需要使用 Ok 去测试一个函数时，运行以下的命令（用函数名代替 FUNCTION）： 
    python3 ok -q FUNCTION --local
    如果你的函数包含了开头为 "DEBUG: 的 print 的调用，那么 OK 程序就会忽略这一行（否则，包含额外的 print 调用会由于额外的输出而导致测试失败）。 
    print("DEBUG:", x)
    还有更多特性说明在 OK 使用指南。你可以通过 OK 帮助 来快速生成大多数的命令。
    除法、向下取整除法和取模 这些是 Python 3 中的一些除法相关的运算符例子： 真实的除法（十进制）：/ 
    >>> 1 / 5
0.2

>>> 25 / 4
6.25

>>> 4 / 2
2.0

>>> 5 / 0
ZeroDivisionError
    向下取整除法（整数除法）：// 
    >>> 1 // 5 # 会把真实的除法结果截断
0

>>> 25 // 4
6

>>> 4 // 2
2

>>> 5 // 0
ZeroDivision Error
    取模（保留余数）：% 
    >>> 1 % 5
1

>>> 25 % 4
1

>>> 4 % 2
0

>>> 5 % 0
ZeroDivisionError
    当除以 0 的时候 ZeroDivisionError （除 0 错误）会出现。 一个有用的技巧是用 % 检查数字 x 是否可以整除另一个数字 y： 
    x % y == 0
    在这个例子中，我们可以通过 x % 2 == 0 来检查 x 是否是一个偶数
    返回和打印 你定义的大多数函数都需要包含一个 return 语句，用于提供调用该函数的表达式返回值 当 Python 执行一个 return 语句时，函数调用会立即终止。如果 Python 到达了函数的末尾且没有执行 return，那么函数就会返回一个 `None` 相比之下，print 函数常常被用来显示数值。不像 return 语句，当 Python 执行一个 print 调用时，函数不会立即终止 
    def what_prints():
    print('Hello World!')
    return 'Exiting this function.'
    print('This course is awesome!')

>>> what_prints()
Hello World!
`Exiting this function.'
    注意 print 显示的文本没有被括号包裹，但是 return 将保留括号
